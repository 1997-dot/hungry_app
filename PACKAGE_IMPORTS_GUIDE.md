## FEATURE LAYERS - Data Layer

### All Data Models (e.g., `user_model.dart`, `product_model.dart`)
```dart

import 'package:equatable/equatable.dart';
// Internal imports
import '../../domain/entities/[entity_name].dart';

```

## Notes

- **Generated files**: Files ending with `.g.dart` or `.config.dart` are auto-generated by build_runner
- **Part files**: Bloc events/states use the `part` keyword to split files
- **Injectable**: `@injectable` and `@lazySingleton` annotations will be added during implementation
- **Internal imports**: Always use relative paths for internal project files
